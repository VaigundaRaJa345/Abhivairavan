# Abhivairavan RetailOS

**Abhivairavan x Zoink digital designs**

A premium retail management operating system for sanitaryware dealers.

## Features
- **Secure Authentication**: Built with NextAuth.js and RBAC.
- **Branch Portals**: Mobile-friendly data entry for branch managers.
- **Master Analytics**: Comprehensive dashboard for admins with KPI tracking and visual charts.
- **Google Sheets Integration**: Serverless database using the Google Sheets API.

## Getting Started
1. Install dependencies: `npm install`
2. Follow the setup guide in `README_SETUP.md`.
3. Start the development server: `npm run dev`

## Deployment
Deploy to Vercel with ease by configuring the required environment variables.
